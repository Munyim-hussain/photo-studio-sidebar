# 📸 Photo Studio Sidebar Menu

A modern, dark-themed sidebar navigation UI for a photo studio website. Built using pure HTML and CSS with a clean layout and smooth design.

---

## 🚀 Features

* 🌙 Dark theme UI
* 📱 Responsive sidebar navigation
* 🎯 Clean and minimal design
* ⚡ Lightweight (no JavaScript)
* 🎨 Smooth hover effects
